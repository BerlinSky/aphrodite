# Starter Kit for Gulp, ES6, Browser-sync 

### Features and Components
1. Compile sass
1. Compile JS
1. Transpile JS ES6 / ES7
1. Autoprefixer
1. Browser-sync 
1. ESLint
1. Browerify 
1. Gulp-uglify
1. gulp

### Steps to use the starter kit

##### Clone the project

##### Install the NPM packages:
  ```
  npm install
  ```

##### Start the app by running gulp:
```
gulp
```
